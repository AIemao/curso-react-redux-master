# Curso de React Full-Stack 🚀

[![React](https://img.shields.io/badge/React-18+-61DAFB?logo=react&logoColor=white)](https://react.dev/)
[![Node.js](https://img.shields.io/badge/Node.js-18+-339933?logo=node.js&logoColor=white)](https://nodejs.org/)

Repositório completo do curso de desenvolvimento web moderno com React, Redux, Node.js, MongoDB e tecnologias relacionadas. Inclui 30 seções de conceitos fundamentais, exercícios práticos e duas aplicações full-stack.

## 🚀 O Que Você Vai Aprender

- ✅ Configuração do Webpack moderno
- ✅ Arquitetura de componentes React
- ✅ Gerenciamento de estado com Redux
- ✅ Desenvolvimento full-stack (MERN stack)
- ✅ Fundamentos do Next.js (SSR, SSG, Rotas)
- ✅ Integração com Firebase/Firestore
- ✅ Sistemas de autenticação
- ✅ Deploy no Heroku
- ✅ Boas práticas para aplicações em produção
- ✅ Operações CRUD com APIs REST
- ✅ Integração TailwindCSS + TypeScript

## 📚 Estrutura do Curso

### 🧠 Fundamentos Essenciais
1. Configuração do Ambiente (VS Code, Node, MongoDB)
2. Conceitos React (Componentes, JSX, Props)
3. React Router DOM
4. Padrões com Hooks Modernos
5. Gerenciamento de Estado com Redux
6. Configuração do Webpack 4+
7. Fundamentos de TypeScript
8. Integração com TailwindCSS

### 💻 Projetos Principais
- **Calculadora** (Fundamentos React)
- **Sistema de Cadastro de Produtos** (Firebase/Firestore)
- **Aplicativo Todo List** (Full-Stack MERN)
- **Sistema de Ciclos de Pagamento** (Integração Redux)
- **Sistema de Autenticação** (JWT, Middlewares)
- **Portal de Cadastro de Usuários** (Next.js + TS)

### ⚙️ Backend
- APIs REST com Node.js & Express
- Operações CRUD no MongoDB
- Middlewares de autenticação
- Boas práticas de segurança em APIs
- Configuração de deploy no Heroku

### 🚀 Tópicos Avançados
- Implementação do Redux-Form
- Server-Side Rendering (Next.js)
- Migração de sistemas legados
- Otimização de performance
- Conceitos básicos de CI/CD

## ⚙️ Instalação

### Pré-requisitos
- Node.js 18+
- MongoDB 7+
- VS Code (Recomendado)
- Postman (Teste de APIs)

```bash
# Clonar repositório
git clone https://github.com/aiemao/react-fullstack-course.git

# Instalar dependências
cd react-fullstack-course && npm install

# Iniciar servidor de desenvolvimento
npm run dev
````
